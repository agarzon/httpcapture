# Security Policy

## Supported Versions

Only the **latest release** is supported with security updates.

## Reporting a Vulnerability

**Please do NOT open a public issue for security vulnerabilities.**

Instead, use **GitHub Security Advisories** to report them privately:

1. Go to the repository's **Settings > Security > Advisories**.
2. Click **New draft security advisory**.
3. Fill in the details and submit.

This ensures the vulnerability is handled confidentially before any public disclosure.

## Response

We will review reports on a **best-effort** basis and aim to respond as quickly as possible. Once confirmed, we will work on a fix and coordinate disclosure.

Thank you for helping keep HTTPCapture secure.
